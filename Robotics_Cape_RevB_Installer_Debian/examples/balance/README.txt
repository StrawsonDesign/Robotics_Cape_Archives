Reference solution for balancing the BeagleMIP, a segway-style
self balancing robot that uses the following robotics cape features:

-accelerometers and gyros (IMU)
-H-bridges for driving motors
-quadrature encoders for wheel position measurement
-reading dsm2 rc radio channels
-sending and receiving mavlink packets
-listen for button presses to exit cleanly
-managing program flow with global system state
